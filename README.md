# Semantic Segmentation
### Introduction
In this project, you'll label the pixels of a road in images using a Fully Convolutional Network (FCN).

### Hyperparameters
learning_rate = 0.0005
epochs = 10
batch_size = 29
keep_prob = 0.8
kernal_init_std = .001
